# Grammar Transformer Project

This is a Python web application that transforms grammar input using defined logic.

## Features
- Grammar transformation
- Web interface
- Python backend

## Technologies Used
- Python
- Flask
- HTML/CSS
- JavaScript

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the application:
python app.py

3. Open in browser:
http://127.0.0.1:5000
